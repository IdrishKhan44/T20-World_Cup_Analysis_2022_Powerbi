# T20-World_Cup_Analysis_2022_Powerbi
T20 World Cup 2022 Analysis is an interactive Microsoft Power BI dashboard project that analyzes player performance, batting, bowling, and team selection using data visualization, DAX, and Power Query. It provides insights through interactive charts, KPIs, filters, and performance comparisons.

# 🏏 T20 World Cup 2022 Analysis – Power BI Project

## 📌 Project Overview

This project is an interactive Microsoft Power BI dashboard developed to analyze player performances and team combinations from the ICC Men’s T20 World Cup 2022. The dashboard provides data-driven insights into batting, bowling, and all-round performances using advanced visual analytics and interactive reporting techniques.

The main objective of this project is to identify the best-performing players in different cricket roles and build an ideal playing XI based on performance metrics.

---

## 🎯 Objectives

* Analyze player performances across the tournament
* Compare batting and bowling statistics
* Identify top performers by role
* Create an ideal team selection using data analytics
* Build an interactive and user-friendly dashboard

---

## 📊 Dashboard Features

* Interactive filters and slicers
* Player performance comparison
* Batting and bowling analysis
* KPI indicators and scorecards
* Role-based player categorization
* Final team selection dashboard
* Dynamic charts and visualizations

---

## 🧠 Key Insights

The dashboard evaluates players using important cricket metrics such as:

* Batting Average
* Strike Rate
* Boundary Percentage
* Economy Rate
* Wickets Taken
* Bowling Average
* Match Consistency

Players are categorized into:

* Openers
* Anchors
* Finishers
* All-Rounders
* Fast Bowlers
